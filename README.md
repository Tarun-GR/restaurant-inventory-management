# restaurant-inventory-management
AI-Powered Restaurant Inventory & Analytics System

=> This system connects restaurant sales data with ingredient-level inventory tracking.
- Each dish is mapped to detailed ingredient quantities, allowing automatic stock deduction and smarter purchase planning.
- The system also includes AI-based features to predict:
- Peak order hours
- Trending dishes
- Sales forecasts
- Personalized dish recommendations
